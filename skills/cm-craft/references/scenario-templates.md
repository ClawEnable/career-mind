# Scenario Templates

Templates for each artifact scenario type. These guide content selection and organization.

## Resume

### Structure

```
# {Name}
{Contact Info}

## Summary
{2-3 sentences tailored to target direction}

## Skills
{Aggregated from entry tags, grouped by category}

## Experience
### {Company} — {Role} ({Period})
- {Bullet 1: strongest impact}
- {Bullet 2: technical depth}
- {Bullet 3: scope/leadership}

## Education
{From profile}
```

### Selection Rules
- 2-4 bullets per role, prioritizing quantified results
- If target_direction set, emphasize matching entries
- Maximum 2 pages — trim weakest entries
- Skills section: aggregate unique tags, group by domain

## Performance Review

### Structure

```
# Performance Self-Review — {Period}

## Summary
{Overall narrative of contributions and growth}

## Key Accomplishments
### {Area 1}
- {Impact statement with metrics}
- {Context and approach}

### {Area 2}
...

## Growth Areas
- {What was learned}
- {How it applies going forward}

## Goals for Next Period
- {Based on target direction and gaps}
```

### Selection Rules
- Organize by impact area, not by project
- Include both outcomes AND growth/learning
- Connect to stated or implied goals
- Professional self-assessment tone

## Promotion Case

### Structure

```
# Promotion Case: {Current Level} → {Target Level}

## Summary
{Why this promotion is warranted — impact narrative}

## Evidence of Level-Above Behavior
### {Criterion 1}
- {Evidence from entries}
- {Scope and impact}

### {Criterion 2}
...

## Scope and Influence
- {Cross-team impact}
- {Mentoring/leadership}
- {Technical decisions and their outcomes}

## Growth Trajectory
- {Progression over time}
```

### Selection Rules
- Lead with impact, not activities
- Demonstrate behaviors expected at target level
- Include scope metrics (team size, user impact, revenue)
- Address promotion criteria explicitly

## Skill Map

### Structure

```
# Skill Map — {Date}

## Technical Skills
### {Domain 1}: {Level}
- {Skill 1}: {evidence summary}
- {Skill 2}: {evidence summary}

### {Domain 2}: {Level}
...

## Leadership & Communication
- {Skill}: {evidence}

## Gaps (vs. target direction)
- {Missing skill 1}
- {Missing skill 2}
```

### Selection Rules
- Aggregate all tags, group by domain
- Rate depth: "Exploring" (1 entry), "Proficient" (2-3 entries), "Strong" (4+ entries with depth)
- Compare against target_direction for gap analysis
- Include soft skills when evidenced in entries
