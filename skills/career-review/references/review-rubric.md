# Review Scoring Rubric

Each dimension is scored on a 3-level scale: **Weak** / **Adequate** / **Strong**.

## Scoring Standards

### 1. Readability

| Level | Criteria |
|-------|----------|
| Weak | Dense wall of text; no visual hierarchy; key info buried; ATS-unfriendly headings |
| Adequate | Basic structure exists; some formatting; key points findable with effort |
| Strong | 3-second scan reveals key strengths; clear headings; consistent formatting; ATS-compatible section names |

### 2. Content Specificity

| Level | Criteria |
|-------|----------|
| Weak | Only responsibilities listed ("responsible for X development"); no actions or outcomes |
| Adequate | Mix of responsibilities and some specific actions; outcomes partially expressed |
| Strong | Every entry has specific action + specific result; reader understands what was achieved and how |

### 3. Quantification

| Level | Criteria |
|-------|----------|
| Weak | No metrics anywhere; "improved efficiency" / "enhanced experience" without data |
| Adequate | Some entries have numbers but lack comparison baselines ("improved 20%" without "from what") |
| Strong | Most entries have metrics with baselines ("from 5min to 30sec", "coverage from 60% to 95%") |

### 4. Agency

| Level | Criteria |
|-------|----------|
| Weak | Passive constructions: "was assigned" / "participated in"; weak verbs: "responsible for" / "assisted with" |
| Adequate | Some active verbs but inconsistent; mix of "led" and "participated" without clear ownership |
| Strong | Strong active verbs: "led" / "designed" / "implemented" / "optimized"; clear ownership of outcomes |

### 5. Narrative Coherence

| Level | Criteria |
|-------|----------|
| Weak | Timeline gaps unexplained; role descriptions contradict level; no clear career direction |
| Adequate | Timeline mostly continuous; career direction somewhat clear but not compelling |
| Strong | Each role builds on previous; career narrative shows growth; timeline is clean with explanations for any gaps |

## Issue Classification

After scoring, classify each issue as:

- **[fixable]** — tailor can fix this: wording, format, structure, weak verbs
- **[needs-extract]** — needs more material: vague experiences, missing quantification, insufficient depth. Specify which experience section needs extraction
