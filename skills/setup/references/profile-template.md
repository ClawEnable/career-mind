# Profile Template

`.career/profile.md` stores the user's personal and professional identity.

## File Format

```markdown
# Profile

## Personal
- Name: {full name}
- Phone: {phone number}
- Email: {email address}
- City: {city}

## Education
- {degree} from {institution}, {year}

## Additional (captured over time from any skill)
- {certifications, languages, other relevant info}
```

## Update Rules

- Any skill may APPEND new information to this file
- No skill may OVERWRITE or DELETE existing content
- When new profile info is discovered during capture or other skills, offer to append it here
- Add new lines to the existing file, never rewrite the entire file
